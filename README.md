# Crop-Recommendation-Platform
AI-powered crop recommendation prompt Library
## Objective
The Crop Recommendation Platform is an AI-based system that helps farmers select the most suitable crop based on environmental and soil conditions. The system analyzes user inputs such as soil type, temperature, rainfall, humidity, season, and location to recommend the best crop. It also provides the reason for the recommendation, suitable fertilizer, water requirement, and expected yield.
## Prompt Engineering Techniques Used
1. Role and Context
2. Few-Shot Prompting
3. Structured Output
4. Chain-of-Thought Prompting
5. Multi-Step Chaining
